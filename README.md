# Personal Profile App

This is a QA Portfolio web application built as a modern react project showcasing quality assurance expertise, certifications, and projects. The application features a React frontend with shadcn/ui components designed to demonstrate professional QA engineering capabilities through an interactive portfolio interface.


### Steps to run in local:
#### Dev Env
- `npm install`
- `npm run dev`

#### Prod Env
- `npm install`
- `npm build`
- `npm start`
